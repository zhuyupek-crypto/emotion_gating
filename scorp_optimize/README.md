## 已完成实验

1. **实验一（低开区间）**：-3%~-4% 为唯一通过 -15% 回撤门槛的区间
2. **实验二（Slots 上限）**：Slots=2 为收益与回撤的最优平衡点
3. **实验三（仓位加权）**：所有加仓方案均突破回撤门槛，1.0 倍仓位最优
