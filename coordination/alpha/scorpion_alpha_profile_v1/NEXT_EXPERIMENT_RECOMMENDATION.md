# 下一实验推荐

## 分析方法

基于169笔纯bear基线的相邻样本分析，按照以下标准筛选:
1. 总EV为正
2. 至少3个两年区间EV为正
3. 不依赖单笔极端收益
4. 样本数量有实际增加

## 推荐结果

**第一项单变量A/B实验: 放宽 低开区间 到 A3_-3_to_-2**

- 样本数: 483
- EV: 0.95%
- 胜率: 53.5%
- 盈亏比: 1.35
- 两年区间EV:
  - 2018-2019: 0.16%
  - 2020-2021: 2.15%
  - 2022-2023: 0.55%
  - 2024-2025: 1.73%
- 最大连续亏损: 14

**注意**: 这是研究筛选推荐，不是自动通过标准。实施前需要:
1. 修改策略中对应的条件（仅此一项）
2. 运行完整回测验证
3. 确认169笔基线交易不受影响
4. 使用影子交易器验证新增交易


## 不推荐的方向

- 低开区间 A1_-6_to_-5: 负EV (EV=-1.19%, n=100)
- 低开区间 A2_-5_to_-4: 负EV (EV=-0.36%, n=199)
- 60日位置 B1_50_to_60: 负EV (EV=-0.22%, n=69)
- 60日位置 B3_70_plus: 负EV (EV=-0.67%, n=393)
